Conocimiento Técnico

Glosario de Inteligencia Artificial

ML
Machine Learning / Aprendizaje Automático
Aprende a base de datos y ejemplos. En lugar de darle órdenes fijas, la computadora descubre patrones por sí misma para predecir o clasificar nueva información.
Fundamentos IA
DL
Deep Learning / Aprendizaje Profundo
Resuelve problemas ultra complejos usando neuronas artificiales. Evolución del ML que procesa imágenes, audio y texto a nivel masivo — tecnología detrás del reconocimiento facial.
Redes Neuronales
GenAI
Generative AI / IA Generativa
Crea contenido nuevo y original. A partir de un prompt genera texto, imágenes, videos, música o código de programación que antes no existían.
Creación de Contenido
NLP
Natural Language Processing / PLN
Entiende y habla el idioma humano. Permite a las máquinas leer, interpretar, traducir y redactar textos en lenguaje natural.
Lenguaje
LLM
Large Language Model / Gran Modelo de Lenguaje
Predice la siguiente palabra en un texto. NLP gigante entrenado con casi todo internet — tecnología que hace funcionar a ChatGPT, Gemini y Claude.
Modelos de Lenguaje
CV
Computer Vision / Visión por Computadora
Ve y analiza el mundo físico. Permite a las máquinas procesar imágenes y videos para identificar objetos, detectar rostros o guiar autos autónomos.
Visión Artificial
RL
Reinforcement Learning / Aprendizaje por Refuerzo
Aprende a base de premios y castigos. Interactúa con un entorno mediante prueba y error — si lo hace bien recibe un premio matemático. Muy usado en robótica.
Entornos Dinámicos
RAG
Retrieval-Augmented Generation
Busca información externa en tiempo real antes de responder. Conecta al LLM con bases de datos externas para generar respuestas precisas sin alucinaciones.
